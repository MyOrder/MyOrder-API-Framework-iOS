MOFFavoritesDAO Class Reference

Inherits from MOFBaseDAO : NSObject
Declared in MOFFavoritesDAO.h

Overview

DAO for managing favorites. Current implementation of favorites is stored locally on device, but API is ready for future server side stored favorites. Any MOFModelObject implementing NSCopy protocol can be favorited

MOFModelObject Class Reference

Inherits from NSObject
Conforms to NSCoding
Declared in MOFModelObject.h

Overview

Basic model class for the MOFFramework. It provides methods to fetch object and persist them based on the MOFModelObjectPersistence selected per class. Persisted objects in any way require an object identifier

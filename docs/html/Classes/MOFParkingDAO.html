- (MOFNetworkConnection *)checkoutParkingSession:(MOFParkingSession *)session licensePlate:(NSString *)licensePlate animation:(MOFNetworkAnimation)animation onResult:(MOFBaseDAOResultBlock)resultBlock onError:(MOFBaseDAOErrorBlock)errorBlock

createParkingSessionWithLocation:orPointId:endTime:animation:onResult:onError:

Creates a new parking session for a location or point id, ending on a particular time.

- (MOFNetworkConnection *)createParkingSessionWithLocation:(CLLocationCoordinate2D)location orPointId:(NSString *)pointId endTime:(NSDate *)date animation:(MOFNetworkAnimation)animation onResult:(MOFBaseDAOResultBlock)resultBlock onError:(MOFBaseDAOErrorBlock)errorBlock

Parameters

location

Location for the new parking session. If kCLLocationCoordinate2DInvalid, then it uses pointId

pointId

Point id corresponding to the parking spot numbers.

date

End date of parking session. Note that it has to end within 24 hours.

animation

Animation type to pass to the animation handler of the request

resultBlock

Block executed with the created MOFParkingSession

errorBlock

Block executed with the resulting error if any.

Return Value

Connection fired. Cancellable.

Discussion

Creates a new parking session for a location or point id, ending on a particular time.
